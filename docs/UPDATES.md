# Updating a live nclawzero device

Current public update paths are Raspberry Pi SD-card reflashes and
container-image updates for:

- `ghcr.io/perlowja/nclawzero-demo`
- `ghcr.io/perlowja/nclawzero-agent`

NVIDIA Jetson family update workflows are deferred pending hardware validation.
Do not use older A/B slot, L4T, or device-mode recovery notes as public
supported procedures.
